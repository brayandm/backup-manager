# Backup Manager

## Setup Git Hooks

```bash
cp .hooks/* .git/hooks/
```

## How to run

1 - [Install Docker Engine](https://docs.docker.com/engine/install/)

2 - Run script:

```bash
curl -o install.sh https://raw.githubusercontent.com/brayandm/backup-manager/main/install.sh && chmod +x install.sh && ./install.sh
```
